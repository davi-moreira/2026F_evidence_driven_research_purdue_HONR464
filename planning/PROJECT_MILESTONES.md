# PROJECT MILESTONES — HONR 46400 Fall 2026 (v2: M0–M15)

Five students, five **individual** projects, one semester-long chain: every
milestone is a graded, submittable artifact with **develop → present → submit →
revise** cadence (exceptions: M13 due the Sunday after Thanksgiving; M10 and
M15 are **terminal** — no revision window). Every milestone updates the
cumulative **Research Project Dossier** and requires an **AI Research Ledger**
entry. Student-facing briefs + rubrics live one file per milestone in
`_research_project/2026Fall/milestone_NN_*.md` (Brightspace replicates from
there); machine anchors in `course_config.yaml milestones:`; validated by
`scripts/validate_milestones.py`. The v1 M00–M23 system is preserved at git
tag `v1-compass-build`.

**Kickoff cadence.** Every milestone is kicked off from its Brightspace brief
at the opening of a Friday studio's sprint block, developed across Mon/Wed
lectures (project-transfer moments) and the studio sprint, red-teamed by peers
+ an assigned GenAI Studio reviewer role where designated (M5, M7, M9, M13),
and submitted at studio close. Kickoff exceptions: M0 (assigned in class on
day 1), M11 (kicked off Monday of conference-prep week), M13 (packages
exchanged at the Week-13 studio), M14 (brief posted with the async module).

Meeting numbers `M1…M43` below are MEETINGS (calendar backbone); milestone IDs
are `M0…M15`.

## Dossier mapping

Each milestone finalizes dossier components: charter (M0–M3), evidence map
(M2), MIDA declaration (M3–M7), data + measurement documentation (M4–M6),
declared analysis protocol (M7), reproducible Colab notebook (M8), claim–
evidence table (M8–M9), robustness + diagnostic record (M9), poster +
presentation materials (M10–M12), replication record (M13), research note
(M14), research chapter + AI-management portfolio (M15). The **AI Research
Ledger** accumulates across all sixteen.

## Notes

- The Expo presentation (Tue Nov 17) is a graded component of M12; there is no
  separate milestone ID for the event itself.
- M13's package exchange is anonymized: each student receives a peer's
  reproducibility package with identifying headers stripped by the instructor.
- Revision policy (syllabus): all milestones except M10 and M15 are revisable
  within 7 days of feedback for up to half the lost points.
- Hard external anchors: URC abstract internal gate Fri Oct 9 (external
  deadline TBD); poster print submission Fri Nov 6 5:00 PM; Expo Tue Nov 17.
